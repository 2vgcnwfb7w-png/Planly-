# CLAUDE.md — Planly Development Guide

## Mission

Build **Planly**, a premium iOS productivity app using SwiftUI.

## Core Rules

- Write production-quality code only.
- Use MVVM.
- Use SwiftData.
- Never use placeholder code unless unavoidable.
- Continue automatically if output is too long.

## UI

- Apple Human Interface Guidelines.
- Premium glassmorphism.
- Smooth animations.
- Dark & Light Mode.
- SF Pro typography.
- Consistent spacing and reusable components.
